# Hop – Jeu Java avec interface graphique

## Description

Hop est un jeu développé en Java utilisant la bibliothèque Swing pour l’interface graphique. Le joueur contrôle un personnage qui saute de bloc en bloc, évite les obstacles et progresse en altitude pour augmenter son score et atteindre de nouveaux niveaux.

Ce projet utilise les principes de la programmation orientée objet et la gestion d’événements clavier.

---

## Fonctionnalités

- Interface graphique avec Java Swing
- Contrôle du personnage avec le clavier
- Gestion de la gravité et du saut
- Détection des collisions avec les blocs
- Système de score basé sur l’altitude
- Progression par niveaux
- Affichage du score et du niveau
- Écran de fin de jeu (Game Over)

---

## Contrôles

- Flèche gauche : déplacer à gauche
- Flèche droite : déplacer à droite
- Flèche haut : sauter
- Flèche bas : plonger

---

## Structure du projet

- `Hop.java` : classe principale, lance le jeu
- `GamePanel.java` : gestion de l’interface graphique
- `Axel.java` : gestion du personnage
- `Block.java` : gestion des blocs
- `Field.java` : gestion du terrain et des niveaux

---

## Technologies utilisées

- Java
- Java Swing
- Programmation orientée objet
